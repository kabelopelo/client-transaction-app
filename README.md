# client-transaction-app

A REST API that allow for creating, updating, deleting and searching for a client and transaction details.

How to run the app:

Run web-api: src/main/java/com/client/transaction/webapi/WebApiApplication.java

Once launched, all end-point can be looked at: http://localhost:8080/app/swagger-ui

Run web-app: /client-transaction-app/web-app> npm install
             /client-transaction-app/web-app> ng serve
